# Digital Twin Notes

## Building Stock Twin

A building-stock digital twin combines location, geometry, age, use type,
systems, metered energy, retrofit history, and scenario assumptions. The twin is
most useful when users can inspect data lineage and uncertainty.

## Local First Analytics

Local-first workflows are useful for portfolio experimentation because data,
prompts, embeddings, generated SQL, retrieved context, and model outputs can be
audited without cloud services.

## Grounded Answers

Grounded answers should state whether they used database rows, document chunks,
or both. They should distinguish synthetic demo data from measured real-world
data and avoid unsupported engineering claims.
